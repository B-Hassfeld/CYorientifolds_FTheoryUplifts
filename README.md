This is a repository for analyzing Calabi-Yau orientifold hypersurfaces and their F-Theory uplifts.
There are two python scripts, one is importing the other, which should be used in addition to CYtools.
One file contains helpful functions whereas the other contains two classes: A Calabi-Yau orientifold class and a F-Theory uplift class.
A demo notebook demonstrates basic usage of the two classes.
